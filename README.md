# RAG random-advice-generator
Receive a piece of advice with just a click of a button.

Try out the app at https://random-advice-generator.herokuapp.com

## Tech Stack
* Frontend
  * HTML
  * Bootstrap
* Backend
  * Python (Flask)
* Deployment
  * Heroku

## Acknowledgements
