# RAG random-advice-generator
Receive a piece of advice with just a click of a button.

Visit us at https://random-advice-generator.herokuapp.com/

## Tech Stack
* Front-End
  * HTML
  * Bootstrap
  * Jinja
* Back-end
  * Flask
* Deployment
  * Heroku
